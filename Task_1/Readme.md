# Español: Task 1 - Web Scraping

**Objetivos:**
- Utilizar bibliotecas de Python como BeautifulSoup o Scrapy para extraer datos de sitios web.
- Identificar y recopilar conjuntos de datos relevantes de páginas web públicas.
- Si no tienes conocimientos de programación, utiliza herramientas automatizadas como Octoparse o ParseHub.
- Aprender a manejar la estructura HTML y la navegación web para obtener datos precisos.
- Crear conjuntos de datos personalizados adaptados a necesidades de análisis específicas.

**Descripción:**

En esta tarea, realizamos el web scraping de la página de la biblioteca Escaramuza utilizando la biblioteca BeautifulSoup de Python. Primero, antes de usar el sitio web, verificamos su disponibilidad y si tenía restricciones. Comenzamos instalando la biblioteca, luego procedimos con el web scraping y revisamos la información que contenía. Después, evaluamos el tipo de información para determinar qué datos extraer para nuestro dataframe. Luego extrajimos la información seleccionada y creamos el dataframe. Posteriormente, analizamos el tamaño del dataframe. Finalmente, lo guardamos como un archivo CSV en este repositorio de GitHub.
