# Español: Task 2 - Exploratory Data Analysis (EDA).

**Objetivos:**
- Formular preguntas relevantes sobre el conjunto de datos antes del análisis
- Explorar la estructura de los datos, incluyendo: variables y tipo de datos
- Identificar tendencias, patrones y anomalías en los datos
- Probar hipótesis y validar suposiciones mediante estadísticas y visualizaciones
- Detectae posibles problemas o datos que deban abordarse en análisis posteriores.

**Descripción:**

Comenzamos importando el archivo CSV creado en la tarea 1. Evaluamos el tamaño del dataframe y información importante, como el nombre de las columnas, la cantidad de datos que contiene cada una y el tipo de datos en cada columna. A partir de esto, determinamos las anomalías e hipótesis a probar en esta tarea. Comenzamos evaluando los datos faltantes en la columna "Autor", luego pasamos a la división y los cambios en la columna "Precio". También evaluamos la información mostrada en la columna "Disponibilidad" para determinar si su formato era correcto. Además, verificamos si había duplicados en más de una columna, ya que esto podría generar confusión en análisis posteriores. Finalmente, evaluamos el formato y la escritura en la columna "Autor" para asegurar que fuera consistente en todos los libros del dataframe. Finalmente, lo guardamos como un archivo CSV en este repositorio de GitHub.
